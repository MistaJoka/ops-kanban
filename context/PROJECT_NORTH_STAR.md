# Project North Star

## Product identity

`ops-kanban` is an AI-first operations board for a solo or small landscaping business.

It is not a generic Trello clone.

The product is a business operating workspace where each job moves through a real-world pipeline.

## Core thesis

```txt
The board is the business.
The card is the source of truth.
Columns are job states.
Views are lenses over the same data.
AI is the operational copilot.
```

## Primary user

A solo landscaper or small landscaping operator who needs a simple, mobile-first system to manage:

- New inquiries
- Site visits
- Estimates
- Scheduling
- Job execution
- Photos and notes
- Invoices
- Payments
- Follow-ups
- Archived work

## Product feel

The app should feel:

- Fast
- Mobile-first
- Operational
- Industrial
- Modern
- Clear for a non-technical user
- Powerful without looking complicated

## MVP promise

A user should be able to run the essential job lifecycle from one universal board.

## Non-goals

Do not turn this into:

- A generic project management app
- A bloated CRM
- A fake demo board
- A multi-board maze
- A dashboard that replaces the Kanban workspace
- A system that requires the user to understand technical concepts

## Primitive compression

```txt
Customer asks.
Job card is born.
Card moves.
Work gets done.
Money gets collected.
AI helps.
Nothing gets lost.
```
